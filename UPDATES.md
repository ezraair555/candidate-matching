## 📝 New Feature: Notes Viewer Page

### Dedicated Notes Page
**URL:** `https://ezraair555.github.io/candidate-matching/notes.html`

**Features:**
- ✅ View ALL saved notes in one place
- ✅ Sorted by newest first
- ✅ Full candidate profile displayed with each note:
  - Name, title, company
  - Location, years of experience
  - Email contact
  - Skills list
  - Rating (stars)
  - Note text
  - Timestamp
  
### Statistics Dashboard
Shows at the top of notes page:
- **Total Notes** - Count of all saved notes
- **Candidates with Notes** - Unique candidates reviewed
- **Average Rating** - Mean rating across rated candidates

### Export Options
From notes page:
- **📊 Export CSV** - Download all notes as spreadsheet
- **📄 Export JSON** - Download structured data
- **🗑️ Clear All Notes** - Bulk delete (with confirmation)

### CSV Fields Exported
```
Candidate ID, First Name, Last Name, Email, Current Title,
Company, Location, Years Experience, Skills, Rating, Note, Created At
```

## 💡 Usage Tips

### Adding Notes
1. Go to main app
2. Select jobs and find matching candidates
3. Click on a candidate card
4. Add note in modal + optional rating
5. Click "Save Note"

### Viewing Notes
1. Click "📝 View All Notes" from header
2. Browse all notes sorted by date
3. See full candidate context with each note
4. Export or clear as needed

### Best Practices
- Rate candidates consistently (1-5 stars)
- Write detailed notes for future reference
- Export notes weekly as backup
- Clear old notes after hiring decisions

## 📊 What's Captured in Notes

Each note includes:
- **Candidate Data:**
  - Full name
  - Current title & company
  - Email address
  - Location
  - Years of experience
  - Full skills list
  
- **Note Data:**
  - Note text (your comments)
  - Rating (0-5 stars)
  - Created timestamp
  - Updated timestamp

All data exports include these same fields for consistency.
